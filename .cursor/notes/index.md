# Invoke-Civitai Converter Notes Index

## Project Overview
Converts InvokeAI-generated PNG images to Automatic1111 format for Civitai upload.

## Project Structure
- `converter.py` - Main conversion script
- `invokeai_cfg.json` - Configuration (model folders, output paths)
- `hash_cache.json` - Cache for model hash calculations
- `converter.bat` - Windows batch script
